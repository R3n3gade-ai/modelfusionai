Model Merging Guide

Welcome to the Model Merger! This tool combines different AI models to create new, more powerful models. Here's everything you need to know:

Quick Start

  1. Select Models: Choose 2+ models from the library on the left
  2. Pick Method: Choose how to combine them (see methods below)
  3. Adjust Settings: Fine-tune with advanced settings if needed
  4. Start Merge: Click "Start Merge" and watch the magic happen!

Merge Methods Explained

Linear (Model Soup)

Easy

What it does: Simply averages the models together

Best for: Similar models, general improvements

When to use: Your first merge, combining related models

SLERP Easy

What it does: Smoothly blends two models like mixing colors

Best for: Two high-quality models

When to use: Want smooth transition between model capabilities

NuSLERP Medium

What it does: Enhanced SLERP with better control

Best for: Fine-tuned models with specific skills

When to use: Need more control than basic SLERP

TIES Advanced

What it does: Resolves conflicts when merging multiple models

Best for: 3+ models, conflicting capabilities

When to use: Merging many specialized models

DARE Advanced

What it does: Randomly drops some changes to reduce interference

Best for: Multiple fine-tuned models

When to use: Models are interfering with each other

Arcee Fusion Expert

What it does: Intelligently identifies and keeps the best features

Best for: High-quality models with distinct strengths

When to use: Want the absolute best from two models

Passthrough Expert

What it does: Stacks model layers to create larger models

Best for: Creating bigger, more capable models

When to use: Want more parameters and complexity

MOE Expert

What it does: Creates a system that routes to different experts

Best for: Specialized models for different tasks

When to use: Want models to specialize in different areas

Advanced Settings

Density

How much of each model to keep (0.3 = keep 30%, 0.7 = keep 70%)

Higher = More of the model, Lower = Less interference

Weight

How much influence each model has in the final result

Higher = More influence, Lower = Less influence

Base Model

The reference model that others are compared to

Usually your best or most stable model

Threshold

How selective to be about which features to keep

Higher = More selective, Lower = Keep more features

Common Issues

Models won't merge: Check they have the same architecture
Poor results: Try adjusting density/weight settings
Merge takes too long: Some methods are more complex than others
Need base model: Some methods require selecting a base model
